Portfolio

Below you will find some examples of coding projects that I have completed in conjunction with the UNC program. Aside from the group projects at the top, the list is in reverse chronological order so newer projects are located toward the top to help demonstrate my progression.

a

IVY

IVY was created in our final group project. IVY is an app designed to help the user plan a road trip without a particular destination in mind. It was our first real exposure to creating an app using React.

Proof

Proof was the product of a unique group project that was built to help whiskey connoisseurs everywhere. Proof was to demonstrate our groups use of handlebars and APIs. It allows the user to search different brands, recipes and locations for enjoying a refreshing cocktail. We were unable to find suitable APIs to use for this project so we pre-populated the databases and incorporated crowd sourcing to expand the databases. Proof is for the people and by the people!

Gold Circle

Gold circle is an app utilizing multiple APIs to assist the user in planning a trip to see his favorite artist. It uses the Spotify, Songkick and Google APIs to assist the user in selecting an event and learning more about the venue and surrounding area.

Memory Game

This homework assignment created a memory game using React. The assignment required the application’s UI to be broken up into different UI components (which managed the component state, and responded to the user’s events).

Burger Time

This homework assignment created a burger logger using MySQL, Node, Express, Handlebars and a homemade ORM and is hosted using Heroku.

Friend Finder

In this homework assignment we had to build a compatibility-based "Friend Finder" application aka a dating app. This full-stack site will take in results from the users' responses to the survey questions and then compare their answers with those from other users. The app will then display the name and picture of the user with the best overall match. Express was used to handle the routing.

Bamazon

This homework assignment taught our class how to incorporate MySQL and Inquirer into our CLI App. The homework called for the creation of an app named Bamazon (similar to Amazon beta 1.0) which will take orders, deplete the inventory and notify the customer of a successful or unsuccessful purchase.

LIRI

The homework called for the creation of an app named LIRI (kind of like SIRI). LIRI is a Language Interpretation and Recognition Interface. LIRI works as a command line node app that takes in parameters inputted by the user and then retrieves the data. LIRI will search Spotify for songs, Bands in Town for concerts, and OMDB for movies.

Train Schedule

This homework called for the creation of a train schedule application that incorporates Firebase to host arrival and departure data. The app retrieves and manipulates this information with Moment.js, providing up-to-date information about various trains, namely their arrival times and how many minutes remain until they arrive at their station.

GoT Gify

This homework called for an app that returned GIFs (with a rating) on the click of a button. It also required a form field where the user could create new buttons that would add to the array thus creating a new search button. I choose to make my app related to game of thrones families/characters because I never know who belongs in which family when watching the show and thought this might be a fun way to help others learn who is in which family or just look up characters.

Movie Trivia

This homework assignment created a Trivia game using JavaScript for the logic and jQuery to manipulate HTML. The game is multiple choice and the user is trying to beat the clock before time runs out.

Crystal Game

This homework assignment created another fun and interactive game for web browsers by dynamically updating the HTML pages with the jQuery library. A random number is generated and the user must try to collect the crystals needed to equal that random number. At the beginning of each game a random value for each crystal is generated and the user must remember this value while trying to match the randomly generated number without going over.

Psychic Game

This homework assignment created a Psychic game that runs in the browser featuring dynamically updated HTML and CSS powered by JavaScript code. The game is as simple as picking a letter and hoping it is the one the program is thinking of. It tracks the number of wins losses and guesses left.

Responsive Portfolio

This homework assignment built on the basic portfolio by introducing our class to bootstrap.

Basic Portfolio

This homework assignment was created using HTML and CSS and also taught the importance of basic web layout principles.
